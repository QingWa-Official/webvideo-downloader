1. chrome 浏览器安装篡改猴插件 https://chrome.google.com/webstore/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo

2. 点击 插件-管理面板-已安装脚本旁的加号 将压缩文件中的 自动下载油猴脚本青蛙改版.js.txt 的全部内容复制粘贴进去，并ctrl+s保存

3. 脚本开头的配置有说明，可以启用或者停用自动刷新下载，刷新时间等，修改好后保存

// 控制是否自动刷新并下载
auto_start: false,
t_start_down: 5000, // 页面刷新后开始下载的时间 单位毫秒
t_click_next: 1500, // 开始下载后到点击下一集的时间 单位毫秒
t_reload_page: 1500, // 点击下一集到刷新页面的时间 单位毫秒
end_num: null,  // 自动下载到多少集(包含)

适用腾讯视频
将开头false改成true
刷新腾讯视频的页面
然后就视频就会5s后自动下载，之后1.5s点击下一集，再过1.5s刷新页面开始下一轮循环


4. 双击运行 自动下载服务器.exe 

5. 打开你要下载的腾讯视频网页，刷新后并等待自动下载

完事大吉，等待视频自动下载吧！


注意后续可以自行修改脚本配置，需要保存脚本+刷新视频网站页面，新的配置才会生效
